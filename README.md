# 🇬🇧 English

## About

**Engineering Lab** is a collection of interactive web simulations focused on engineering, mechanics, automation and industrial technology.

The goal is to explain technical principles visually and interactively directly in the browser.

Each project is built as a standalone mini simulation using:

- HTML
- CSS
- JavaScript
- Canvas / SVG

No external libraries are required unless stated otherwise.

## O projektu

**Engineering Lab** je kolekce interaktivních webových simulací zaměřených na strojírenství, mechaniku, automatizaci a průmyslovou techniku.

Cílem projektu je vysvětlovat technické principy názorně, vizuálně a interaktivně přímo v prohlížeči.

Každý projekt je samostatná mini simulace vytvořená pomocí:

- HTML
